# Laporan Tugas 1: Website Personal

## 1. Deskripsi Proyek
Proyek ini adalah website personal statis yang dirancang untuk memenuhi persyaratan Tugas #1. Tujuan dari website ini adalah untuk menampilkan portofolio, profil diri, dan menyediakan formulir kontak. 

**Fitur Utama:**
- Beranda dengan profil singkat.
- Halaman portofolio dengan tampilan grid.
- Halaman formulir kontak.
- Desain responsif untuk berbagai ukuran layar.
- Toggle mode gelap (Dark Mode).
- Navigasi yang dapat diakses melalui keyboard.

**Teknologi yang Digunakan:**
- HTML5 Semantik
- CSS3 (Flexbox & Grid, CSS Variables)
- FontAwesome (Ikon)
- Google Fonts (Inter)

## 2. Struktur Folder dan File
```text
/
├── assets/
│   └── images/
│       └── favicon.ico       (Ikon untuk tab browser - perlu ditambahkan)
├── style/
│   └── main.css              (File styling utama)
├── index.html                (Halaman utama)
├── portfolio.html            (Halaman karya/portofolio)
├── contact.html              (Halaman kontak)
└── LAPORAN.md                (Dokumentasi)
```
